# Code of Conduct

## Our Pledge
We pledge to make participation in our project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Being respectful
- Accepting constructive feedback

## Enforcement Responsibilities
Project maintainers are responsible for clarifying standards.

## Scope
This Code of Conduct applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported.

## Enforcement Guidelines
Maintainers will follow guidelines in determining consequences.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
